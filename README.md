# TikTok Claims Data Analysis

**Python • pandas • Data Inspection • EDA Preparation • PACE Framework**

## Overview

This project demonstrates the early stages of a data analytics workflow using Python and pandas. Working within a fictional TikTok data team scenario, the objective was to inspect, organize, and summarize a claims classification dataset before exploratory data analysis and machine learning.

## Scenario

TikTok is developing a machine learning model to classify videos as either claims or opinions. Before modeling can begin, the dataset must be inspected to understand the available variables, check data quality, and identify useful features.

## Skills Demonstrated

- Python

- Data inspection
- Boolean filtering
- Grouping and aggregation
- Descriptive statistics
- Feature creation
- PACE Framework

## Key Tasks Completed

- Imported and loaded the TikTok dataset
- Reviewed dataframe structure with `head()`, `info()`, and `describe()`
- Examined claim status distribution
- Compared engagement metrics for claims and opinions
- Analyzed author ban status
- Created engagement-rate variables
- Prepared findings for future EDA and modeling

## Key Takeaway

This project shows how Python and pandas can be used to inspect and organize data before deeper analysis. Early data inspection helps identify useful variables, potential issues, and next steps for building a reliable machine learning model.
